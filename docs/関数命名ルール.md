# 関数命名ルール

関数名は「動詞 + 名詞」で意味が分かる命名にしてください。
略語・曖昧な動詞は使わないでください。

---

## 種類別の命名規則

| 種類 | 規則 | 例 |
|------|------|----|
| Sub（アクション） | 動詞 + 名詞 | `DeleteSheetIfExists`, `ApplyHeaderStyle` |
| Function（値を返す） | 動詞 + 名詞 | `GetLastRow`, `ExtractTableName` |
| Boolean Function | `Is` / `Has` + 名詞 | `IsEmptyRange`, `HasBom` |
| Private ヘルパー | 同上 + `Private` で宣言 | `ParseColumnLine`, `BuildDataArray` |

---

## 推奨動詞一覧

| 動詞 | 用途 |
|------|------|
| `Load` | ファイル・データを読み込む |
| `Parse` | テキスト・構造を解析する |
| `Extract` | 文字列・要素を取り出す |
| `Build` | 配列・文字列を組み立てる |
| `Write` | Excelシートへ書き出す |
| `Apply` | 書式・設定を適用する |
| `Get` | 値・オブジェクトを取得して返す |
| `Create` | シート・オブジェクトを新規作成する |
| `Delete` | シート・データを削除する |
| `Select` | ユーザーに選択させる |
| `Validate` | データの妥当性を確認する |
| `Format` | 表示形式を整える |
| `Count` | 件数を数える |

---

## 命名例

```
LoadSqlFile          … SQLファイルを読み込む
ParseCreateTable     … CREATE TABLE ブロックを解析する
ExtractBacktickName  … バッククォートで囲まれた名前を取り出す
BuildColumnArray     … カラム情報を2D配列に組み立てる
WriteTableDefinition … テーブル定義をシートへ書き出す
ApplyTableHeaderStyle … テーブルヘッダー行に書式を適用する
GetLastRow           … 最終行番号を返す
CreateOutputSheet    … 出力シートを新規作成する
DeleteSheetIfExists  … 指定名のシートが存在すれば削除する
SelectSqlFile        … ファイル選択ダイアログを表示してパスを返す
IsEmptyRange         … 範囲が空かどうかを返す
CountUniqueTables    … 重複を除いたテーブル数を返す
```

---

## 禁止パターン

| パターン | 理由 | 修正例 |
|---------|------|-------|
| 略語：`GetTbl` | 意味が不明 | `GetTableName` |
| 曖昧な動詞：`DoProcess`, `Run`, `Exec` | 何をするか不明 | 具体的な動詞を使う |
| 連番：`Process1`, `Process2` | 処理内容で命名 | `ParseHeader`, `ParseBody` |
| 動詞なし：`TableName`, `FileName` | 動作が不明 | `GetTableName`, `SelectFile` |
