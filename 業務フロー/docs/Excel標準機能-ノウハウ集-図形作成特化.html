クイックアクセスツールバー
  • 01 設定画面が開いたら左上のプルダウン「コマンドの選択」を 「すべてのコマンド」 に変更する
    ⚠️ ここを「すべてのコマンド」にしないと目的の機能が一覧に出てこないことが多い。これが最重要。
  • 02 左側の一覧から登録したい機能を探す → 選択して 「追加 >>」 ボタンを押す
  • 03 右側の一覧（登録済み）で 上下ボタン を使って順番を並び替える
    左から1番目が Alt+1、2番目が Alt+2 に対応するので、使用頻度が高い順に並べる
  • 04 OK を押して完了。Excelを再起動しなくても即反映される。
💡 登録後は Alt+数字で即起動：QATの左から1番目が Alt+1、2番目が Alt+2 と自動対応します。Altキーを押すとExcelがQAT上に数字のバッジを表示してくれるので、何番かを覚えなくてもその場で確認できます。

設定グリッドにスナップ — 図形が自動でセル境界に吸着する
▼
図形を選択した状態で ›
図形の書式タブ ›
配置グループ ›
配置 ▼ → グリッドの設定
「グリッドにスナップ」をONにすると、図形をドラッグしたときにセルの境界線に自動で吸着します。手動での微調整が不要になり、図形の位置が自然に揃います。

「図形にスナップ」もONにすると、他の図形の端にも吸着するようになります。複数の図形を並べるとき、上端・左端がピタッと揃います。

一時的にスナップを無効にしたいときは、ドラッグ中に Alt キーを押し続けると、その間だけスナップが外れてピクセル単位で自由に移動できます。

モードカーソルの見た目操作使いどころ
通常モード 白い十字（セル選択） クリック → セルが選択される テキスト入力・数式
オブジェクト選択モード 矢印カーソル ドラッグ → 範囲内の図形だけ選択 図形を複数まとめて選ぶとき ✅
⚠️ Ctrl+A との違い：Ctrl+A はシート上のすべての図形（ヘッダー帯・仕切り線含む）を選択します。オブジェクト選択モードでドラッグすると、ドラッグした矩形の内側にある図形だけが選択されます。スイムレーン本体だけを選びたいときはこちらを使います。
✅ 終了方法：Escキーを押すか、もう一度「オブジェクトの選択」をクリックすると通常モードに戻ります。QATに登録しておくと ON/OFF が1クリックで切り替えられます。
選択Tabキーで図形を1個ずつ順番に選択する
▼
Tab → 次の図形へ移動（シート上の図形を順番に選択）
Shift+Tab → 前の図形へ移動
コネクタ（矢印線）だけを選びたいときに特に有効です。コネクタはクリックで選ぼうとしても図形の裏に隠れていて選びにくいのですが、Tabキーで順番に巡回すると確実に選択できます。

使い方：まずシート上の何か図形を1つクリックし、そこからTabで次々と移動します。目的の図形が選択されたところで操作（書式変更・テキスト編集など）を行います。
💡 Tabキーでの選択はコネクタ・図形・テキストボックスを区別なく順番に巡回します。図形の追加順（シートに配置した順）で移動するので、後から追加したものは後の方に出てきます。

書式の統一

コードなしで書式を揃える3つのアプローチ

書式書式ブラシ（連続適用） — ダブルクリックで複数図形に連続転写
▼
ホームタブ ›
クリップボードグループ ›
書式のコピー/貼り付け（ブラシアイコン）
  • 01 コピー元にしたい図形を シングルクリック で選択する（ここはシングルで良い）
  • 02 ブラシアイコンを ダブルクリック → カーソルがブラシ型に変わったら連続モード開始
  • 03 書式を貼り付けたい図形を次々と クリック していく（シート内のどこにある図形でも可）
  • 04 Esc を押して終了 → 通常カーソルに戻る
クリック回数挙動使いどころ
1回クリック1図形に貼り付けて自動終了1個だけ揃えたいとき
ダブルクリックEscまで連続貼り付け複数図形をまとめて揃えるとき ✅
⚠️ コピーされるのは「書式だけ」です。フォント・色・サイズ・枠線が揃います。テキスト内容はコピーされません。コネクタ（矢印線）にも適用でき、線色・太さ・矢印スタイルが揃います。
💡 空白セルをクリックしても連続モードは終了しません。意図せず適用することはないので、図形を探しながら落ち着いてクリックできます。
書式既定の図形スタイルを設定 — 新しく置く図形が最初から統一される
▼
  • 01 見た目を基準にしたい図形（フォント・色・枠線が設定済みのもの）を右クリック
  • 02 コンテキストメニューから 「既定の図形に設定」 を選ぶ
  • 03 以降、そのシートに新しく図形を挿入すると最初からその書式で作成される
✅ これが最も根本的な解決策です。「新しく置くたびに書式が崩れる」原因は、Excelが最後に設定した書式を記憶していないためです。既定スタイルを設定しておくと、置いた直後から書式が揃った状態で始まります。フロー作成の最初にこの設定をするだけで、書式の後追い修正がほぼ不要になります。
適用範囲：この設定はブック単位で記憶されます。同じブックを開き直しても設定は保持されます。ただし別のブックには引き継がれないため、新しいブックで作業を始めるときに毎回設定します。

F4 または Ctrl+1 → 図形を選択した状態で押すと「図形の書式設定」パネルが開く
図形を右クリック ›
図形の書式設定 ›
「サイズとプロパティ」タブ（□アイコン）
高さ・幅・位置をcm単位で直接入力できます。例えば「すべての処理図形を幅2.5cm・高さ1.2cmに統一する」場合、1つを設定してから残りに書式ブラシで転写、もしくは複数選択してまとめて入力します。

複数選択時の注意：複数の図形を選択した状態でサイズを入力すると、すべての図形が同じサイズになります。ただし位置（左・上）の入力は最初の図形を基準に相対移動する場合があるため、位置の一括変更は配置メニューを使う方が安全です。
整列グループ化 — スイムレーン単位でまとめて移動・コピー
▼
Ctrl+G → グループ化（複数図形を選択した状態で）
Ctrl+Shift+G → グループ解除
  • 01 オブジェクト選択モードで1スイムレーン分の図形をドラッグ選択
  • 02 Ctrl+G でグループ化 → レーン全体が1つの図形として扱われる
  • 03 グループのまま移動・コピー・位置調整が可能
  • 04 グループ内の個別図形を編集したい場合は グループをダブルクリック → 内部の個別図形をクリック
💡 グループのネストも可能：スイムレーン1本をグループ化したものを、さらに複数まとめてグループ化できます。「列Aのグループ」「列Bのグループ」をまとめて「フロー全体グループ」にすることで、フロー全体の移動が1回の操作でできます。

テキスト入力の効率化

ダブルクリックより速いキーボードだけの入力フロー

入力F2キーで図形のテキスト編集モードに入る
▼
図形を選択した状態で： F2 → テキスト編集モードに入る（ダブルクリックと同じ効果）
Esc → テキスト編集を終了して図形選択状態に戻る
Esc もう一度 → 図形の選択も解除してシートに戻る
マウスを使わずに図形のテキストを編集する流れ：
Tabキーで目的の図形を選択 → F2でテキスト編集モード → テキストを入力 → Escで編集終了 → Tabで次の図形へ移動 → 繰り返す。

この流れをマスターするとマウスをほぼ使わずにテキスト入力が完了します。
入力Tab → F2 の連続フロー — キーボードだけで全図形を入力する
▼
  • 01 最初の図形を1回クリックして選択する（最初だけマウスを使う）
  • 02 F2 → テキスト編集モードに入る
  • 03 テキストを入力する
  • 04 Esc → 編集終了（図形が選択された状態のまま）
  • 05 Tab → 次の図形に移動
  • 06 02〜05を繰り返す
✅ 最初のクリック以外はすべてキーボードで完結します。図形のダブルクリックが不要になるため、手がマウスとキーボードを行き来する動作がなくなり体感速度が大きく上がります。
💡 テキスト入力中の改行：図形のテキスト編集中に Enter を押すと改行されます（セルの確定ではなく改行）。図形から抜け出したいときは Esc を使います。
